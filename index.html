홈 화면 기록 방식 A/B 비교 — 도뚜 골라줘!

A 퀵 입력

체중 + 생리만 바로 보이고, 나머지는 "더 기록하기" 눌러서 펼침. 저장 버튼은 아래에 일반 배치.

  • 기본: 체중 + 생리량 2개만
  • "＋ 더 기록하기" → 기분/증상/메모 펼침
  • 저장 버튼: 페이지 하단 (스크롤 필요할 수 있음)
  • ✅ 매일 체중만 적을 때 빠름
💡 80% 상황(체중만 기록)에 최적화
🅰️ A안 보기 →

B 퀵 입력 + 플로팅 저장

A안과 동일하게 퀵 입력, 대신 저장 버튼이 화면 하단에 항상 떠있어서 스크롤 없이 바로 저장.

  • 기본: 체중 + 생리량 2개만
  • "＋ 더 기록하기" → 기분/증상/메모 펼침
  • 저장 버튼: 항상 하단에 고정 (플로팅)
  • ✅ 확장해도 저장 버튼 항상 보임
💡 기분/증상까지 기록할 때도 편함
